En este repositorio se documentan las pruebas realizadas para la página Demon Blaze, enfocadas en dos historias de usuario clave:

Historia de Usuario 4: Validación del flujo de compra, incluyendo la interacción con el carro de compras. Se verificaron funcionalidades como la adición, modificación y eliminación de productos, así como la confirmación del proceso de pago.
Tambien podemos visualizar desde  Google drive con el enlace : https://drive.google.com/drive/folders/17IplZLe9suG-vXHyQ5vxVth34alNF0Au?usp=sharing

Historia de Usuario 5: Pruebas exhaustivas de un formulario dentro de la plataforma, asegurando el correcto manejo de validaciones de entrada, mensajes de error y envío exitoso de la información.
Tambien podemos visualizar desde  Google drive con el enlace :https://drive.google.com/drive/folders/14GW1_knKkNza-CAjXMhdiuovDIXtBwgq?usp=sharing
Ambas historias se trabajaron con un enfoque de QA manual, cubriendo desde la creación de casos de prueba hasta la recolección de evidencias y reporte de bugs, garantizando una experiencia de usuario fluida y sin errores.
